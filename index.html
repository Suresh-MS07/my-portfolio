Phonebook Application

Project Goal

To develop a robust and efficient console-based application for managing contacts, focusing on core programming concepts and data persistence in C.

Key Features

  • Add, list, modify, search, and delete contact records.
  • Data is saved to a file (`.dat`), ensuring persistence between sessions.
  • Clean, menu-driven interface for easy navigation in the console.

Challenges & Solutions

A key challenge was managing file I/O operations safely to prevent data corruption. I implemented structured data writing and reading, along with temporary files for modification and deletion to ensure data integrity.

Tech Stack

C Language, Standard I/O Libraries, File Handling
